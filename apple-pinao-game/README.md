# 钢琴块红球游戏

一款简单有趣的点击类休闲小游戏，玩家需要点击下落的红球得分，在倒计时结束前尽可能获得更高分数。



[TOC]

## 游戏介绍

### 核心玩法

- 游戏初始倒计时为 20 秒
- 点击屏幕中的红球即可得分，每次点击 + 1 分
- 点击红球后所有红球会向下移动一行，同时顶部随机生成新的红球
- 红球超出游戏区域后会自动消失
- 倒计时结束后游戏结束，展示最终得分

### 游戏规则

- 每个红球仅可点击一次，点击后会有视觉反馈
- 游戏开始后倒计时开始，无法暂停
- 红球下落有平滑的动画效果
- 游戏结束后展示最终得分

## 技术实现

### 核心特性

1. 响应式动画：使用 CSS transition 和 `requestAnimationFrame` 实现流畅的红球下落动画
2. 精确计时：使用 `performance.now ()` 实现高精度倒计时，确保时间计算准确
3. 内存管理：完善的定时器清理和 DOM 元素销毁机制，防止内存泄漏
4. 容错处理：DOM 元素获取增加容错机制，避免页面崩溃
5. 兼容性处理：兼容旧版浏览器的 `Element.closest` 方法

### 关键配置（config.js）

| 常量      | 值    | 说明             |
| --------- | ----- | ---------------- |
| ROWS      | 6     | 游戏区域行数     |
| COLS      | 4     | 游戏区域列数     |
| CELL_SIZE | 120   | 单元格尺寸（px） |
| GAP       | 2     | 单元格间距（px） |
| INIT_TIME | 20.00 | 初始倒计时（秒） |



## 运行方式

1. 将项目文件放入 Web 服务器根目录（如 Nginx、Apache）
2. 直接打开 index.html 文件（部分浏览器可能因 ES6 模块限制需要服务器环境）
3. 点击页面中的红球开始游戏
